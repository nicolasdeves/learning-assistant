# Learning Assistant

## Getting started

### Backend

- npm start run

- npx prisma generate

- npx prisma migrate dev

- npx prisma migrate dev --name `migration_name`

- npx prisma db seed

- npx prisma migrate reset

### Mobile

- npm start 

- npm run android

- npx react-native start --reset-cache

- ./gradlew clean

### Rodar no celular

- adb devices (certificar que vรก aparecer somente o celular - fechar o emulador)

- npx react-native run-android --deviceId RX8M7135Q2B (o que foi exibido no comando anterior)
